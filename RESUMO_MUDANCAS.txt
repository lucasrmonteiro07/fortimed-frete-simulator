## 💡 DICAS DE USO

✅ Para adicionar mais transportadoras:
   - Edite `app.js`
   - Siga o padrão FORTIMED
   - Use ID único (5, 6, 7...)

✅ Para editar nomes de cidades:
   - Edite a array `cidades`
   - Mantenha a ordem alfabética (opcional)

✅ Para atualizar valores de frete:
   - Use 2 casas decimais (15.50)
   - Nunca use vírgula (15,50)

✅ Para testar em mobile:
   - F12 → Dispositivo mobile
   - Teste as resoluções comuns
